# Agent Instructions for Xcoder

You are an expert iOS Engineer. You must operate within this project without using the Xcode GUI.

### Rules of Engagement
1. **Source Code**: All SwiftUI files must reside in `Sources/XcoderApp/`.
2. **Project Structure**: Never attempt to edit `.xcodeproj` files. If you add files or change settings, edit `project.yml` and run `xcodegen generate`.
3. **Build & Deploy**: 
   - To push to TestFlight: Run `bundle exec fastlane preview`.
   - To debug on a local device: Run `bundle exec fastlane local`.
4. **Self-Correction**: If `xcodebuild` returns an error, analyze the logs, fix the Swift code in `Sources/`, and retry the build automatically.

### Project Context
- **Primary Language**: Swift / SwiftUI
- **Target Platform**: iOS 16.0+
